# SQL Reports

## Purpose
This folder contains SQL queries used to generate HR compensation,
payroll, and analytical reports.

## Types of Reports
- Department-wise salary summaries
- Role-based compensation analysis
- Monthly and annual payroll reports
- Cost and trend analysis for HR leadership

## Files
- sql_reports.sql: Core SQL queries used for reporting and analytics

## Notes
All reports are validated using sample data and QA test cases
to ensure accuracy and consistency.
